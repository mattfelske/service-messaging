# service-messaging
A simple REST API service that allows the user to perform CRUD operations on a messages database collection.

# service-user-management
A Service for building and maintaining User Profiles.

### Installation
_Please note if you have issues, contact the development team for the proper schema._

1. git clone the repo.
2. From the project root directory `npm install`
3. Execute `npm run start`

The endpoints are available within the json and are importable to Postman
